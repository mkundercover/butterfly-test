# The Link - AR Butterfly Experience 🦋
Esperienza di Realtà Aumentata Web-based.

## 🚀 Tecnologie utilizzate
- **A-Frame**: Web framework per la realtà virtuale.
- **AR.js**: Augmented Reality per il web.
- **Three.js**: Engine grafico 3D.

## 📁 Struttura Progetto
- `/assets`: Modelli 3D e risorse grafiche.
- `/css`: Stili per l'interfaccia utente.
- `/js`: Logica dello sciame e gestione sensori.

## 📱 Come utilizzarlo
Inquadra il qr code davanti a te, premi "START" e posiziona il dispositivo in verticale.
